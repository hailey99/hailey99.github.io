remove odd values in any properties
```javascript
function removeOddValues(obj) {
  for (let prop in obj) {
    if (obj[prop] % 2 !== 0) {
      delete obj[prop];
    }
  }
  return obj;
}
```

array 안의 숫자 평균
```Javascript
function computeAverageOfNumbers(nums) {
  let len = nums.length;
  if (len === 0) {return 0;}
  let sum = 0;
  for (let i = 0; i < len; i++) {
    sum = sum + nums[i];
  }
  return sum/len;
}
```
